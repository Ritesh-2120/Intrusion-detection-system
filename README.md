# Intrusion Detection System using Machine Learning

This project is a Machine Learning-based Intrusion Detection System (IDS) developed using the **NSL-KDD dataset**. It identifies and classifies network intrusions into normal and attack classes using multiple ML algorithms.

---

##  Features
- Data preprocessing and encoding for NSL-KDD dataset  
- Model training using multiple algorithms  
- Comparison of model accuracy  

---

##  Algorithms Used
- Random Forest  
- XGBoost  
- K-Nearest Neighbors (KNN)

---

##  Dataset
**Dataset:** NSL-KDD  

---

##  Requirements
To install all dependencies, run:
```bash
pip install -r requirements.txt
```

---
## Run the Project 
```bash
jupyter notebook intrusion_detection.ipynb
```
